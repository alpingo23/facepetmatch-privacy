Privacy Policy - Face Pet Match

This Privacy Policy explains how the Face Pet Match application (hereinafter referred to as the "Application") processes user data.

1. Data Collected

The Application does not collect or store users' personal data or photos. Photos are used temporarily for facial analysis purposes using the face-api.js library on our servers. No user data or photos are stored permanently or linked to a specific identity.

2. Camera Access

The Application requires access to the device's camera in order to capture a user's photo for the pet matching feature. This photo is used solely for real-time processing and is not saved on the device or servers. The photo is discarded immediately after the analysis is completed.

3. Data Usage

Captured photos are processed on servers exclusively for the purpose of matching users with pets based on facial similarity. The data is not used for any other purposes and is never retained after processing.

4. Data Sharing

The Application does not share any user data, including photos or analysis results, with third parties. All processing occurs within the scope of the Application.

5. Policy Regarding Children

This Application is not intended for children under the age of 13. We do not knowingly collect personal information from children under 13. As no personal data is stored, any use by children is not tracked or recorded.

6. User Rights

Since the Application does not store any user data, there is no need for data deletion or access requests. For questions or concerns, you can contact us at alpipekciler@gmail.com.

7. Updates

This Privacy Policy may be updated from time to time. Any changes will be announced within the Application.
